Header

The <header> element represents introductory content or a set of navigational links.

Main

The <main> element represents the main content of the document.

Navigation (Nav)

The <nav> element represents a set of navigation links.

Section

The <section> element defines a thematic grouping of content, typically with a heading.

Article

The <article> element represents self-contained content, such as a blog post or news article.

Aside

The <aside> element represents content that is tangentially related to the main content, such as sidebars or callouts.

Details

The <details> element is used for content that users can expand or collapse.

Footer

The <footer> element represents footer content, typically containing information about the document or author.

Dialog

The <dialog> element represents a dialog box or interactive component.

Div

The <div> element is a generic container for grouping and styling content.

Span: The <span> element is an inline container for grouping and styling text or elements.
