This is a Heading 1 (H1)

The <h1> element represents the highest level heading in a document. It is typically used for the main title of a webpage or section.

This is a Heading 2 (H2)

The <h2> element represents a secondary heading. It is often used for sub-sections of the main content.

This is a Heading 3 (H3)

The <h3> element represents a third-level heading, which is often used for smaller subsections of a content block.

This is a Heading 4 (H4)

The <h4> element represents a fourth-level heading, and is typically used for less important subsections.

This is a Heading 5 (H5)

The <h5> element represents a fifth-level heading, which is used for even more granular divisions in content.

This is a Heading 6 (H6)

The <h6> element represents the lowest-level heading, and is typically used for less prominent sections or subheadings within an article or section.
